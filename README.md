# Flask Candidates App

Приложение на Flask для работы с кандидатами.

Запуск
1. Установить Flask: `pip install flask`
2. Запустить приложение: `python app.py`
3. Открыть в браузере: http://127.0.0.1:5000/

## Маршруты
- `/` — список всех кандидатов
- `/candidate/<id>` — страница кандидата
- `/search/<name>` — поиск по имени
- `/skill/<skill>` — поиск по навыку
